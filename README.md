# Simon Says Game

A simple memory-based game with four colored buttons.
The game shows a blinking color sequence and the player must repeat it from the beginning.

## Features
- Four colored buttons
- Random color sequence
- Current score and highest score
- Retry option after game over

## How to Play
1. Press any key button to begin
2. Watch the blinking color
3. Repeat the sequence in the correct order
4. Game ends if a wrong button is pressed
